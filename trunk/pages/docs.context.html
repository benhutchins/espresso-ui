Creating a context menu

Syntax:

new Espresso.Context($(element), menu, options)

Espresso.Context Methods

initialize()
setOpacity()

initialize(selector, items, options) Context menu constructor, used when creating a new context menu (new Espresso.context(...))

selector Element or elements of items to add context menu to

items Context menu list elements

options Hash map of window options, here is the key list:
Key Default Description
class menu desktop Class(es) to add to list container
pageOffset 25 Distance menu can be from edge of window
opacity 1 Context Menu's opacity
zIndex 100 Z-Index of Context Menu, set higher than everything else
children false Let context menu work for selector children or just selector elements
Event callbacks $empty All event callbacks:
onHide onShow onSelect


setOpacity(opacity) Sets menu's opacity

opacity Float value between 0 and 1


Adding a context menu to an element

Example HTML: 
<div id="desktop">Hell world</div>
Example Javascript: 
	if ($('desktop')){
		Espresso.Context($('docsLink'), [
			{
				'name': 'New',
				'callback': function(e) {
					var tagName = e.target.tagName.toLowerCase(), x = e.page.x, y = e.page.y;
					alert('you clicked on <' + tagName + '> element at x: ' + x + ', and y: ' + y);
				}
			},
			{separator: true},
			{
				'name': 'Save',
				'url': 'http://www.xvolter.com/'
			},
			{
				'name': 'Save as',
				'callback': function() {
					alert ('Saving as..');
				}
			}
		]);
	}
