Aspects of Espresso UI

  • Desktop - A simple desktop div that acts as a container for windows
  • Dock - Requires the Desktop, allows windows to minmize to it and an add a startmenu widget
  • Context Menus - Add context menus to elements
  • Windows - Create XWindows within a XHTML web page
  • Dialog - Create dynamic javascript alerts, confirms, and info boxes

Classes

Espresso - Class base (contains global variables for all other classes)
Espresso.Desktop - Create a web desktop
Espresso.Dock - Create a dock/task bar to allow windows to minmize to tray
Espresso.StartMenu - Create a start menu widget
Espresso.Context - Create context menus for elements
Espresso.Window - Create a new window
Espresso.Scrollbar - Add styled scrollbars to elements and windows
Espresso.Dialog.Alert - Create dynamic javascript alerts
Espresso.Dialog.Confirm - Create dynamic javascript confirms
Espresso.Dialog.Info - Create dynamic javascript info boxes
